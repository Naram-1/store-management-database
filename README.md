# Store Management System

This project is a small database system for a store.  
It helps manage products, customers, orders, and stock.

The system uses a MySQL database and a Python program that runs SQL queries to show information from the database.

---

# What the system can do

The program allows the user to:

- View all products in the store
- View orders together with customer names
- See which products are included in orders
- Show the best selling products
- Show products that have low stock
